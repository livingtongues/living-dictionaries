Within a dictionary entry, there is a Notes field where users can stylize with italics and bullet points, etc. However, there are discrepancies between the preview mode and the dictionary entry view. We need both areas to display the same material, stylizing and spacing. For example:

- The bullet points do not display on the preview area but they do display on the dictionary entry
- Line spacing looks fine in the preview but then there is too much spacing on the dictionary entry.
